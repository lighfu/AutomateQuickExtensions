Thank you for using this tool.
In this update, we have added a setting that allows you to disable unused features, and made some minor improvements to the code.
We hope you will continue to support QeFa.

・ Added permission settings for executing features (this allows you to disable features you don't use).
・ Added a way to read the status of the function execution permission settings
・ Added icon.
・ Added option to start Automate when QeFa starts.
・ Improved startup behavior
・ Updated source code and some packages.
